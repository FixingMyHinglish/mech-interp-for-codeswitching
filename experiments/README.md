# New-Compute Experiments

This directory contains standalone runnable experiments:

- `exp4_switch_activation/` - Experiment 4 (switch-point neuron activation patterns)
- `exp6_language_selectivity/` - Experiment 6 (language-selective neurons)
- `exp21_language_pair_comparison/` - Experiment 21 (Hi-En vs Fr-En comparison)
- `exp7_neuron_clustering_interpretation/` - Experiment 7 (neuron clustering + interpretation)

Each experiment has its own `run.py` and writes outputs to its own folder.

Shared helpers live in `common/`.
